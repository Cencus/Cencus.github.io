什么是 OpenCode？

OpenCode 是一个开源的 AI 编程助手，支持多种 AI 模型，可以在终端中自主完成代码编写、重构、审查等任务。

常用快捷键：

快捷键 功能
Ctrl + U 一次性删掉光标到行首的所有内容
Ctrl + A 移动到行首
Ctrl + E 移动到行尾
Enter 发送消息
Tab 切换代理模式（build/plan）
Ctrl+P 打开命令面板
Ctrl+X L 切换会话
Ctrl+X M 切换模型
Ctrl+C 退出

常用参数

参数 用途
run 'prompt' 单次执行并退出
-c / --continue 继续上次会话
-s <id> 恢复指定会话
--model 指定模型
-f <file> 附加文件
--thinking 显示思考过程

查看使用统计

注意事项

  1. 交互式会话需要终端支持（在 Hermes 中使用 pty=true）
  2. 使用 Ctrl+C 退出，不要用 /exit
  3. 并行任务使用不同的工作目录避免冲突
  4. 单次任务用 opencode run 更简单
