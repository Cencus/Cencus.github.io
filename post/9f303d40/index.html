Vim 常用命令速查

模式切换

命令 说明
i 光标前插入
a 光标后插入
o 下方新建行插入
O 上方新建行插入
v 可视模式（选中文本）
V 行可视模式
Esc 退出当前模式

移动

命令 说明
h j k l 左 下 上 右
w / W 下一个单词开头
b / B 上一个单词开头
0 行首
$ 行尾
gg 文件开头
G 文件结尾
:10 跳到第 10 行

删除/复制/粘贴

命令 说明
dd 删除整行
dw 删除到单词结尾
d$ / D 删除到行尾
yy 复制整行
yw 复制单词
p 粘贴到光标后
P 粘贴到光标前
x 删除当前字符

编辑

命令 说明
u 撤销
Ctrl + r 重做
. 重复上一个操作
r 替换当前字符
cw 替换单词
cc 替换整行
>> 缩进
<< 取消缩进

搜索替换

命令 说明
/word 向下搜索
?word 向上搜索
n / N 下/上一个匹配
:s/old/new/g 当前行替换
:%s/old/new/g 全文替换
:%s/old/new/gc 全文替换（逐个确认）

保存退出

命令 说明
:w 保存
:w file 另存为
:q 退出
:q! 强制退出（不保存）
:wq / ZZ 保存并退出
:x 保存并退出（仅修改时保存）

窗口/标签页

命令 说明
:sp 水平分屏
:vsp 垂直分屏
Ctrl + w w 切换窗口
:tabnew 新建标签页
gt / gT 下/上一个标签页

入门技巧： 先记 i（插入）、Esc（退回普通模式）、:wq（保存退出）、dd（删行）、yy + p（复制粘贴）这几个，日常够用了～
