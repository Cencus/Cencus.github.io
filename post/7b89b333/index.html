二、Linux系统信息查看：Kali Linux

关键命令：

  • neofetch - 显示系统信息和ASCII logo
  • uname -a - 查看内核信息
  • free -h - 查看内存使用情况
  • lscpu - 查看CPU详细信息

四、安全实践入门：Hack The Box

Hack The Box (HTB) 是一个在线平台，用于学习和实践网络安全技能。

CTF基础概念

在CTF（Capture The Flag）挑战中，通常需要找到两个标志（flag）：

  • user.txt - 用户权限flag
  • root.txt - root权限flag

示例操作

学习路径：

  1. 信息收集（Reconnaissance）
  2. 漏洞扫描（Vulnerability Assessment）
  3. 漏洞利用（Exploitation）
  4. 权限提升（Privilege Escalation）
  5. 后渗透（Post-Exploitation）

重要提示：

  • 在HTB上练习时要遵守平台规则
  • 不要在真实环境使用未经授权的技术
  • 学习安全知识是为了防御，而非攻击

总结

本文涵盖了四个方面的内容：

  1. 编程入门 - Python基础语法和简单程序设计
  2. 系统认知 - Linux系统信息和虚拟机环境
  3. 命令操作 - 终端基础命令的使用方法
  4. 安全实践 - CTF平台和渗透测试入门概念

建议初学者按顺序学习，先掌握基础命令和编程概念，再深入系统管理和安全领域。实践是最好的老师，多动手操作才能真正理解这些知识。
