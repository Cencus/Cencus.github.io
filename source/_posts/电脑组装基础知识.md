# CPU

CPU是电脑的"大脑"，选择CPU首先要确定平台（AMD还是Intel），然后根据预算和需求选择具体型号。

## AMD vs Intel：怎么选？

| 对比维度 | AMD | Intel |
| :------- | :-- | :---- |
| **性价比** | 整体更高，同价位多核性能强 | 单核性能优势，游戏表现略好 |
| **超频** | 几乎全系支持，B系列主板就能超 | 只有K后缀型号支持，需Z系列主板 |
| **功耗发热** | X3D系列游戏功耗很低 | 近代产品功耗偏高，需要注意散热 |
| **升级空间** | AM5接口承诺支持到2027+ | LGA1700已到尽头，下一代换接口 |
| **适合人群** | 多任务工作者、追求性价比玩家 | 追求极致帧率、直播推流用户 |

**AMD后缀含义：**

| 后缀 | 含义 | 说明 |
| :--- | :--- | :--- |
| **X** | 高性能版 | 默认频率更高，是各系列的"标准"高端型号 |

| **G** | 带核显 | 内置核显，不需要独立显卡就能亮机 |

> **🎮 游戏玩家重点关注 X3D 系列**：如 7800X3D、9800X3D，是目前游戏性能最强的消费级CPU，功耗低、温度低、帧数高。

**Intel后缀含义：**

| 后缀 | 含义 | 说明 |
| :--- | :--- | :--- |
| **K** | 可超频 | 解锁倍频，需搭配Z系列主板才能超 |
| **KF** | 可超频+无核显 | 能超频但没核显，价格稍便宜，必须配独立显卡 |

> **⚠️ Intel 13/14代注意**：i9系列（如14900K）存在缩缸稳定性问题，选购前建议了解最新情况，或直接选择15代（Core Ultra）。

### 挑选机箱前，你需要知道的3个硬指标

无论外观多酷炫，保证配件能装进去是首要前提。你只需要核对以下三个核心数据：

1. **CPU散热器限高**：决定了你能装多高的风冷散热器。如果你用原装散热或一体水冷，基本不用操心。
2. **显卡限长**：**最关键的数据**。比如你之前看的那张微星RTX 5070 Ti，长度约303mm，那么你选的机箱显卡限长必须大于这个数字。
3. **电源限长**：决定了能装多长的电源。普通ATX电源都没问题，但如果是特别长的高功率电源（比如1600W以上）需要留意。

选完版本，接下来就是瓦数。根据你的需求，可以参考下面这张推荐表：

| 你的目标显卡       | 建议电源瓦数                 |
| :----------------- | :--------------------------- |
| RTX 5060 级别      | 额定 **650W** 的足额金牌电源 |
| RTX 5070 / 5070 Ti | 推荐 **750W - 850W**         |
| RTX 5080 级别      | 建议 **850W - 1000W**        |
| RTX 5090 级别      | 建议 **1000W - 1200W**       |

### 如何挑选电源？关键看“安全刚需”

回到你最关心的问题：现在装机，电源到底怎么选？

对于新配的电脑（特别是准备用RTX 5070及以上显卡），强烈建议选择 **ATX 3.1（兼容12V-2×6接口）的电源**。

原因很简单，它多了一道“安全保险”，能确保原生的12V-2×6接口稳定连接，让你用得更安心。虽然一些高品质的ATX 3.0电源可能也能用，但为了避免任何潜在的风险，多花一点预算选择更成熟的ATX 3.1标准，是更省心的选择。

# 主板

**选择主板的核心就是“三步走”：先选平台（AMD还是Intel），再看大小（板型），最后挑品牌**。

| CPU平台   | 定位       | 芯片组型号   | 一句话总结                                                   |
| :-------- | :--------- | :----------- | :----------------------------------------------------------- |
| **AMD**   | **发烧级** | X870 / X870E | 接口拉满，强制支持PCIe 5.0显卡和USB4。                       |
|           | **主流级** | **B850**     | **甜品之选**，性能够用，性价比极高，日常游戏完全足够。       |
|           | **入门级** | B840 / A620  | 精简版，满足基础需求，适合预算有限的办公或入门机。           |
| **Intel** | **发烧级** | **Z890**     | 功能完整，是搭配K系列后缀CPU超频的必备选择。                 |
|           | **主流级** | **B860**     | 主流甜品级，虽不支持CPU超频，但该有的PCIe 5.0和高速内存支持一应俱全。 |

Intel处理器后缀带"K"（如i7-14700K）的才支持超频，必须搭配Z系列主板才能发挥。而绝大多数AMD处理器都支持超频，搭配B系列主板就能超，这也是AMD平台的一个性价比优势。

### 二步：测量“体格”——选定主板板型 (ATX, M-ATX, ITX)

锁定了平台，就可以根据机箱大小和拓展需求，来选择主板的“身材”，也就是板型了。

| 规格                    | 尺寸 (大约)  | 优缺点与适用场景                                             |
| :---------------------- | :----------- | :----------------------------------------------------------- |
| **ATX (标准大板)**      | 305 x 244 mm | **优点**：空间大，插槽多（7个PCIe），散热好，适合安装多显卡、高端声卡。 **缺点**：需要中塔及以上机箱，占据更多桌面空间。 |
| **M-ATX (紧凑小板)**    | 244 x 244 mm | **优点**：**绝大多数人的选择**，尺寸适中，接口足以满足绝大多数玩家需求。 **缺点**：PCIe插槽较少（通常4-6个），对多设备用户稍显局促。 |
| **Mini-ITX (迷你主板)** | 170 x 170 mm | **优点**：极致小巧，可组建高性能迷你主机。 **缺点**：**扩展性很差**，通常只有1个PCIe插槽，基本等于“插上显卡就满了”。 |

选好板型后，和机箱核对一下就好：**支持ATX的机箱能装ATX, M-ATX, ITX主板；支持M-ATX的机箱能装M-ATX, ITX主板；而支持ITX的机箱就几乎只能装ITX主板。**

### 三步：了解“品牌”——选择系列和用料

现在市面上主流的消费级品牌，主要是华硕、微星、技嘉和七彩虹这四家。它们的产品线逻辑都差不多，大多是**旗舰>主流>入门**的梯队。

这里我以华硕和微星为例，帮你理解一下这个产品定位的思路：

| 品牌       | 旗舰系列       | 主流系列              | 入门系列         | 一句话点评                                                   |
| :--------- | :------------- | :-------------------- | :--------------- | :----------------------------------------------------------- |
| **华硕**   | ROG (玩家国度) | TUF GAMING (电竞特工) | PRIME (大师)     | 一线大厂，软件体验好，售后完善。ROG是信仰，TUF是皮实耐用性价比高。 |
| **微星**   | MEG            | MPG                   | MAG / PRO        | 做工用料扎实，MAG“军火库”系列（如B850/B860 TOMAHAWK）广受好评。 |
| **技嘉**   | AORUS          | AORUS ELITE / GAMING  | UD               | 超耐久用料是其特色，对AMD平台的X3D系列CPU有专门优化。        |
| **七彩虹** | iGame          | CVN                   | Battle-Ax (战斧) | 近年来发展很快，iGame旗舰系列堆料猛，CVN系列纯白PCB外观颜值高。 |

> **🧠 快速总结**：其实这几家主流品牌都很成熟，同价位的用料和做工差距不大。对于追求性价比的玩家，**锁定“御三家”的M-ATX规格的B系列主板（比如微星B850M TOMAHAWK或华硕TUF GAMING B850M-PLUS）**，基本就是靠谱的选择，很少会出错。

### 再升华一下：供电相数、PCB层数是什么？

如果还想再深入了解一些，可以去看看评测文章里常提到的“供电相数”和“PCB层数”，它们是衡量用料好坏的进阶指标：

- **供电相数**：可以理解为给CPU输送"电力"的通道。简单来说，通道越多，每一条的"负载"就越小，发热和压力也就更小，主板运作起来也更稳定。通常用"18+2+2相"这种形式表示，**数字越大，通常意味着能更好地支持高性能CPU**。
- **PCB层数**：电路板的层数，可以理解为城市道路的"层数（高架、地面等）"。层数越多，信号干扰越小，内存超频的潜力也越大。**主流为6层，高端为8层或10层**。

# 内存 (RAM)

内存是电脑的"短期记忆"，影响系统流畅度和多任务能力。目前主流是DDR5，DDR4已逐步退出主流市场。

## DDR4 vs DDR5：需要换吗？

| 对比项 | DDR4 | DDR5 |
| :----- | :--- | :--- |
| **频率** | 2133-3600 MHz | 4800-8000+ MHz |
| **单条容量** | 最大32GB | 最大64GB，单条32GB很常见 |
| **电压** | 1.2V | 1.1V，更省电 |
| **兼容性** | 只能插DDR4主板 | 只能插DDR5主板，**不兼容** |
| **价格** | 便宜 | 稍贵，但差价已很小 |

> **🚫 注意**：DDR4和DDR5**物理不兼容**，主板买错就插不进去！选购前务必确认主板支持的内存类型。

## 频率怎么选？越高越好吗？

频率是内存的核心参数，频率越高，数据传输越快。但要注意：

| CPU平台 | 推荐频率 | 说明 |
| :------ | :------- | :--- |
| **AMD Ryzen 7000/9000** | 6000MHz CL30 | 甜点频率，超过6400可能有兼容问题 |
| **Intel 13/14代** | 6400-7200MHz | 高频收益明显，可冲更高频率 |
| **Intel Core Ultra** | 6400-8000MHz | 新平台高频支持更好 |

**时序（CL值）也很重要**：同样频率下，CL值越低越好。比如6000MHz CL30 比 6000MHz CL36 延迟更低。

## 容量选择建议

| 使用场景 | 推荐容量 | 说明 |
| :------- | :------- | :--- |
| **办公/轻度娱乐** | 16GB (8G×2) | 最低门槛，预算有限可选 |
| **主流游戏** | 32GB (16G×2) | **最推荐**，未来几年够用 |
| **3A大作+直播** | 48GB (24G×2) 或 64GB (32G×2) | 大作+后台程序不虚 |
| **生产力/视频剪辑** | 64GB起步 | 大项目必备 |

> **💡 双通道很重要**：两条内存组成双通道，带宽翻倍。16GB选 8G×2 而非单条16G，32GB选 16G×2 而非单条32G。

## 内存品牌推荐

| 档位 | 品牌 | 代表系列 | 特点 |
| :--- | :--- | :------ | :--- |
| **高端** | 芝奇 | Trident Z5 RGB (焰刃/幻锋戟) | 颜值高，超频能力强 |

# 硬盘 (存储)

硬盘分两类：**系统盘**（装系统和常用软件）必须用SSD，**数据盘**（存游戏、视频等）可选SSD或机械硬盘。

## SSD：固态硬盘（必选）

### 接口类型：M.2 NVMe vs SATA

| 接口类型 | 速度 | 形态 | 适合场景 |
| :------- | :--- | :--- | :------- |
| **M.2 NVMe (PCIe 4.0)** | 7000MB/s+ | 条状，直插主板 | **系统盘首选**，速度最快 |
| **M.2 NVMe (PCIe 3.0)** | 3500MB/s | 条状，直插主板 | 老平台或预算有限 |
| **SATA SSD** | 550MB/s | 2.5寸方盒 | 给老电脑升级，或做副盘 |

### SSD选购要点

| 参数 | 建议 |
| :--- | :--- |
| **容量** | 系统盘 **1TB起步**，500GB很快就不够用 |
| **接口** | 优先选 PCIe 4.0 NVMe，PCIe 5.0太贵没必要 |
| **缓存** | 有独立DRAM缓存的更稳定，但无缓HMB技术也够用 |
| **寿命(TBW)** | 数值越大越好，1TB盘至少600TBW |

### SSD品牌推荐

| 档位 | 品牌 | 代表型号 | 特点 |
| :--- | :--- | :------- | :--- |
| **旗舰** | 三星 | 990 Pro | 业界标杆，性能最强 |
| | 西数 | SN850X | 游戏优化，稳定可靠 |
| **主流** | 致态 | TiPlus7100 / TiPro7000 | 国产之光，性价比高 |
| | 铠侠 | RC20 / SD10 | 性价比不错，原厂颗粒 |
| **入门** | 宏碁 | GM7000 | 便宜大碗 |

> **⚠️ 避坑**：尽量选原厂颗粒品牌（三星、西数、铠侠、致态、海力士），杂牌盘可能用白片颗粒，掉速掉盘风险高。

## 机械硬盘 (HDD)：大容量存储

| 场景 | 推荐配置 |
| :--- | :------- |
| **纯游戏/日常** | 不需要机械硬盘，SSD够用 |
| **存大量视频/素材** | 4TB以上机械盘做数据盘 |
| **NAS/冷备份** | 红盘(NAS专用) 或 企业盘 |

机械硬盘品牌主要是西数和希捷，注意区分：
- **蓝盘**：家用普通盘，性价比可以
- **黑盘**：高性能盘，游戏用
- **红盘**：NAS专用，7×24小时运行
- **紫盘**：监控专用
- **企业盘**：可靠性最高，价格也贵

# 散热器

CPU散热器负责把热量从CPU带走。选择散热器主要看：CPU发热量、机箱空间、噪音偏好。

## 风冷 vs 水冷

| 类型 | 优点 | 缺点 | 适合场景 |
| :--- | :-- | :--- | :------- |
| **风冷** | 可靠无泄漏、免维护、便宜 | 体积大、高负载时风扇噪音明显 | 中低端CPU、追求稳定 |
| **一体水冷 (AIO)** | 散热强、颜值高、噪音低 | 有泄漏风险、寿命有限(3-5年) | 高端CPU、追求颜值 |
| **分体水冷** | 性能极致、可定制 | 贵、复杂、需维护 | 发烧友专属 |

## 风冷散热器选购

| 类型 | 适合CPU | 价格参考 |
| :--- | :------ | :------- |
| **下压式** | 65W以下入门CPU（如i3、R5非X） | 50-100元 |
| **单塔4热管** | 100W以下（如7500F、13400F） | 100-200元 |
| **双塔6热管** | 150W以下（如14600K、9700X） | 200-350元 |
| **顶级风冷** | 200W以下（如9800X3D） | 350-600元 |

**主流风冷品牌**：利民（性价比之王）、九州风神、猫头鹰（Noctua，静音天花板）、雅浚

## 一体水冷 (AIO) 选购

| 规格 | 冷排尺寸 | 适合CPU | 风扇数量 |
| :--- | :------- | :------ | :------- |
| **120mm** | 120×120mm | 入门CPU | 1把 |
| **240mm** | 240×120mm | 中端CPU（14600K以下） | 2把 |
| **280mm** | 280×120mm | 中高端CPU（更安静） | 2把 |
| **360mm** | 360×120mm | 高端CPU（9700X/14700K） | 3把 |
| **420mm** | 420×120mm | 旗舰CPU（14900K/9950X） | 3把 |

> **⚠️ 选水冷前确认**：机箱是否支持对应尺寸冷排？前置还是顶置？冷排厚度会影响显卡空间。

**主流AIO品牌**：瓦尔基里（颜值性能兼备）、九州风神、海盗船、NZXT、华硕ROG

## 特殊情况：X3D处理器

AMD X3D系列（如7800X3D、9800X3D）**游戏功耗很低**，不需要顶级散热：
- 7800X3D：双塔风冷或240水冷足够
- 9800X3D：同理，甚至单塔也能压

# 装机顺序建议

了解了各硬件之后，这里给一个推荐的装机顺序和注意事项：

## 标准装机流程

```
1. 准备工作
   └─ 清理桌面，准备螺丝刀、扎带
   └─ 阅读主板说明书，了解接口位置

2. 安装CPU
   └─ 打开CPU插槽拉杆
   └─ 对准三角形标记，轻轻放入（不要用力！）
   └─ 扣上拉杆

3. 安装内存
   └─ 查看主板说明书，确认优先插槽（通常是A2/B2）
   └─ 两端对齐，用力按下直到"咔"一声

4. 安装M.2 SSD
   └─ 找到M.2插槽（通常在CPU下方）
   └─ 斜插SSD，拧紧尾部螺丝

5. 安装CPU散热器
   └─ 先涂硅脂（豌豆大小，不要太多）
   └─ 固定散热器，接好风扇线

6. 安装主板到机箱
   └─ 先安装I/O挡板（如果是独立的）
   └─ 安装铜柱，对应主板螺丝孔位
   └─ 放入主板，拧螺丝（不要拧太紧）

7. 安装电源
   └─ 电源风扇朝向看机箱设计
   └─ 先理好线，再固定

8. 安装显卡
   └─ 拆掉机箱对应挡板
   └─ 对准PCIe插槽插入，听到"咔"声
   └─ 接好显卡供电线

9. 接线
   └─ 主板24pin供电
   └─ CPU 8pin/4+4pin供电
   └─ 前面板跳线（开机、重启、USB、音频）
   └─ 风扇接口

10. 检查与开机
    └─ 检查所有线缆是否接紧
    └─ 第一次开机可能要点两下（内存训练）
```

## 常见避坑提示

| 问题 | 解决方案 |
| :--- | :------- |
| **点不亮** | 检查CPU供电线、内存是否插紧、显示器接显卡而非主板 |
| **风扇狂转但黑屏** | 可能内存兼容问题，尝试单条内存或更新BIOS |
| **机箱USB口不好用** | 检查前面板USB线是否接到主板正确位置 |
| **显卡太长装不下** | 买显卡前务必量机箱显卡限长！水冷前置时特别注意 |
| **散热器挡住内存** | 选散热器时注意"内存限高"参数 |
